# SSH setting  

## 1. rpb setting(server)  
### 1. open SSH  
use terminal insert:  
``$ sudo raspi-config 
``  
### 2. choose interfasing options  
### 3. choose SSH
### 4. choose enable (ok)
### 5. Get rpb's ip.  
use terminal insert:  
``$ ip address 
``  



## 2. my computer(client)  
### 1.use terminal  
``$ ssh user_name@ip_adress
``  
### 2. use MobaXterm (recommend) 
1. Dowonload MobaXterm from web.  
2. use MobaXterm create a SSHsection. (ok)
